- This is a project aiming to create an open-source automation suite, aiming for self-service automation of tech savvy mechanical engineers in R&D environments. It revolves around the following components: Postgres as a backbone, NocoDB as the DB frontend, n8n for workflow automation, metabase for analytics and reporting and uptime kuma for monitoring and alerting. The project documentation will be German as this is aimed at the DACH market.
